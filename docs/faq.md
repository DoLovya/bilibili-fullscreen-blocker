# FAQ

## 为什么扩展无法隐藏“bilibili.com — 要退出全屏，请按 esc”？

该提示属于 Edge/Chromium 浏览器在进入全屏时显示的系统级 UI（安全提示），不属于网页 DOM。内容脚本与油猴脚本只能操作网页 DOM，无法影响浏览器 UI，因此无法隐藏。

## 为什么这个提示会覆盖在开发者工具之上？

浏览器 UI 的渲染层级高于网页内容与开发者工具面板区域。提示覆盖在开发者工具上通常意味着它不在网页 DOM 中。

## 是否可以通过 CSS（例如 :fullscreen）隐藏？

主流 Edge/Chromium 版本中不可行。`:fullscreen` 等 CSS 选择器只能影响网页内容的渲染，无法影响浏览器 UI。

## 有没有替代方案？

- 使用 Bilibili 播放器的“网页全屏”替代浏览器“真全屏”，通常不会触发该提示
- 从浏览器设置/实验选项/企业策略方向尝试抑制浏览器 UI，但可用性随版本变化，且可能影响安全性与兼容性

## 本仓库的扩展还做什么？

保留了 Manifest V3 的示例工程结构与注入入口，用于复现、调试与文档说明，不再承诺可以隐藏浏览器系统级全屏提示。
